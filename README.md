# Mweetwa Mweembo Nketani - Professional Portfolio

A modern, responsive portfolio website showcasing the work and expertise of Mweetwa Mweembo Nketani, Sales Engineer at Bravo Technology (Z) Ltd.

## Features

- **Modern UI/UX Design**: Sophisticated minimalism with enhanced interactions
- **Responsive Design**: Optimized for mobile, tablet, and desktop devices
- **Dark/Light Mode**: Automatic system preference detection with manual toggle
- **Accessibility**: WCAG-compliant with keyboard navigation and screen reader support
- **Performance**: Optimized animations and efficient code structure
- **SEO**: Proper meta tags and semantic HTML structure

## Technologies Used

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern design system with CSS custom properties
- **Vanilla JavaScript**: No framework dependencies for optimal performance
- **Inter Font**: Modern typography for enhanced readability

## Sections

- **Hero**: Professional introduction with call-to-action buttons
- **About**: Technical bridge narrative with company information
- **Skills**: Tabbed interface for technical and sales engineering skills
- **Projects**: Filterable portfolio of professional and academic projects
- **Contact**: Interactive contact form with copy-to-clipboard functionality

## Deployment

This portfolio is designed for GitHub Pages deployment and can be easily hosted on any static web server.
